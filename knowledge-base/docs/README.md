# Knowledge Base Docs

This folder contains operational documentation for installing and using the Knowledge Base package.

## Contents

- [Installation Flows](installation-flows.md)

## Quick Summary

- Install target is `~/.kb`.
- Installer updates VS Code user settings automatically.
- Main settings keys:
  - `chat.agentFilesLocations`
  - `chat.instructionsFilesLocations`
  - `chat.promptFilesLocations`

Use the detailed guide in [Installation Flows](installation-flows.md) for exact commands.
