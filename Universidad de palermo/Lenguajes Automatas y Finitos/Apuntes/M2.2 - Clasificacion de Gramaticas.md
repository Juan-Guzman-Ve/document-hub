## Clasificar Gramáticas — Método Estructurado

---

## 📋 El Método — 4 preguntas en orden

Siempre te hacés estas preguntas **de abajo hacia arriba** (del tipo 3 al tipo 0), porque los tipos más restrictivos son subconjuntos de los menos restrictivos:

```
¿Es Tipo 3? → ¿Es Tipo 2? → ¿Es Tipo 1? → Es Tipo 0
```

### Pregunta 1 — ¿Es Tipo 3?

Todas las reglas deben ser de la forma:

- `A → a` ó `A → aB` (lineal derecha)
- `A → a` ó `A → Ba` (lineal izquierda)
- `S → λ` permitido
- **⚠️ No podés mezclar** lineal derecha con lineal izquierda

### Pregunta 2 — ¿Es Tipo 2?

Si no es Tipo 3, fijate si todas las reglas tienen **un solo no terminal a la izquierda:**

- `A → cualquier cosa` ✓
- `AB → algo` ✗ (dos no terminales a la izquierda → no es Tipo 2)

### Pregunta 3 — ¿Es Tipo 1?

Si no es Tipo 2, fijate si se cumple que **el lado derecho es siempre mayor o igual en longitud al izquierdo:**

- `|lado derecho| ≥ |lado izquierdo|`
- Excepción: `S → λ` permitido si S no aparece a la derecha de ninguna regla

### Pregunta 4 — ¿Es Tipo 0?

Si no cumple ninguna de las anteriores → **Tipo 0**, sin restricciones.

---

## 🔍 Ejemplos resueltos

---

### Ejemplo 1

```
P = { S → bbS | bb }
```

|Regla|Forma|¿Cumple Tipo 3?|
|---|---|---|
|`S → bbS`|`A → aB` con a=bb, B=S|✓ lineal derecha|
|`S → bb`|`A → a` con a=bb|✓|

✅ **Es Tipo 3** — todas lineales por la derecha, no hay mezcla.

---

### Ejemplo 2

```
P = { S → λ | A, A → AA | c }
```

|Regla|Análisis|
|---|---|
|`S → λ`|permitido|
|`S → A`|`A → B` sin terminal → **no es Tipo 3**|
|`A → AA`|lado derecho tiene dos no terminales → **no es Tipo 3**|
|`A → c`|✓ forma válida|

¿Es Tipo 2? → Todas tienen **un solo no terminal a la izquierda** ✓

✅ **Es Tipo 2**

**L(G) = { cⁿ / n ≥ 0 }**

---

### Ejemplo 3

```
P = { S → AB, A → λ | aA, B → b | bB }
```

|Regla|Análisis|
|---|---|
|`S → AB`|lado derecho tiene dos no terminales → no es Tipo 3|
|`A → λ`|permitido|
|`A → aA`|lineal derecha ✓|
|`B → b`|✓|
|`B → bB`|lineal derecha ✓|

¿Es Tipo 2? → Todas tienen **un solo no terminal a la izquierda** ✓

Pero `S → AB` no es ni `A→a` ni `A→aB` → **no es Tipo 3**

✅ **Es Tipo 2**

> 💡 El documento aclara que esta gramática tiene una Tipo 3 equivalente: `P = { S → aS | bB, B → λ | bB }`

---

### Ejemplo 4

```
P = { S → λ | A, A → Ad | cA | c | d }
```

|Regla|Análisis|
|---|---|
|`A → Ad`|`A → Ba` → lineal **izquierda**|
|`A → cA`|`A → aB` → lineal **derecha**|

⚠️ **Mezcla lineal izquierda con lineal derecha** → No es Tipo 3

¿Es Tipo 2? → Todas tienen un solo no terminal a la izquierda ✓

✅ **Es Tipo 2**

---

### Ejemplo 5

```
P = { S → aSc | A, A → bAcc | bcc }
```

|Regla|Análisis|
|---|---|
|`S → aSc`|un no terminal a la izquierda ✓|
|`A → bAcc`|un no terminal a la izquierda ✓|
|`A → bcc`|✓|

No es Tipo 3 (lado derecho tiene más de un símbolo mezclado)

¿Es Tipo 2? → Todas tienen **un solo no terminal a la izquierda** ✓

✅ **Es Tipo 2**

**L(G) = { aⁿ bᵐ c²ᵐ⁺ⁿ / n ≥ 0, m ≥ 1 }**

## 📌 Resumen visual del método

```
¿Un solo NT a la izquierda en TODAS las reglas?
          │
         NO → ¿|der| ≥ |izq| en todas? → NO → Tipo 0
          │           │
          │          SÍ → Tipo 1
          │
         SÍ → ¿Todas son A→a, A→aB o A→Ba SIN mezclar?
                    │
                   SÍ → Tipo 3
                    │
                   NO → Tipo 2
```
